# Memory Assistant

人間的な記憶システムを持つローカルAIアシスタント

## 概要
このプロジェクトは、人間の記憶システム（短期記憶、長期記憶、記憶の圧縮）を
模倣したAIアシスタントです。完全ローカル実行でプライバシーを保護します。

## コンセプト
- 重要な情報を抽出・圧縮して保存
- 目標やスケジュールを管理
- 「忘れっぽいが忠実」な性格
- 音声入力対応

## 技術スタック
- LLM: Ollama (Llama 3.1)
- Backend: Python + FastAPI
- Database: SQLite
- Frontend: Electron + React
- Voice: Whisper
